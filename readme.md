Данный репозиторий содержит исходный код высокопроизводительного In-Memory Key-Value кэша.

Основная логика работы содержится в проекте TcpServer.csproj

В проекта ParsingData.csproj содержится код парсера (CommansParser.cs), а так же код потокобезопасного хранилища данных (SimpleStore.cs)

Для тестироования производительности был написан отдельный парсер, работающий на основе массивов, а не Span. Результаты представлены на скриншоте ниже:
